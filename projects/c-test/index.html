Hilfe Text einfügen Aus Datei exportieren Sammlungen Text entfernen

Diese Webseite hilft euch zu den C-Tests zu vorbereiten. Es generiert C-Test aus jeder Text. Es gilt die Regel, dass Sie immer nur das halbe Wort oder einen Buchstaben mehr ergänzen sollen. Erstes und letztes Sätze sind in der Regel ohne Lücken.

Es gibt drei Wege den Text einzufügen. Klicken Sie auf "Text einfügen" um den Text aus Zwischenablage einzufügen. Sie können auch Text exportieren indem Sie entweder eine Datei auswählen oder ein URL tippen.

Wenn Sie auf "Text entfernen" klicken, wird den C-Test entfernt. Die "Sammlungen" Feature ist noch in Development und wird später verfügbar.
